Meta Token Exchange

Paste a short-lived Meta token to exchange it for a 60-day long-lived token and a never-expiring page token. Requires META_APP_ID and META_APP_SECRET set in Vercel env vars.

Token Exchange Result
How to Get Tokens (Manual Steps)
  1. Go to Graph API Explorer
  2. Select your app, click Generate Access Token
  3. Grant permissions: pages_show_list, pages_read_engagement, pages_read_user_content, instagram_basic, instagram_manage_insights, read_insights, ads_read
  4. Paste the token above and click Exchange Token
  5. Copy the Page Token (never expires) and set it as META_SOCIAL_TOKEN in Vercel
  6. Copy the Long-Lived Token (60 days) and set it as META_MARKETING_TOKEN in Vercel
